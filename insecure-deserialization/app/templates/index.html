Configuration Loader Service

This service accepts YAML configuration files for processing.

API Endpoints

  • POST /api/config — Submit YAML configuration (JSON body with "config" or "config_b64" key, or raw YAML)
  • GET /api/healthcheck — Service health and version info

Example

curl -X POST http://target:5000/api/config \
  -H "Content-Type: application/json" \
  -d '{"config": "name: test\nversion: 1.0"}'
